## Project Overview

Fast-moving consumer goods (FMCG) companies like PepsiCo operate across multiple markets and product lines, making consistent performance tracking difficult. This dashboard was developed to:

- Integrate and visualize sales, volume, and profitability data
- Provide region and product level analysis
- Support decision-making with intuitive interactive views
- Highlight trends and anomalies in performance

## Features

- Dynamic visualizations of revenue, profit, units sold, and trend analysis
- Region-wise and product category dashboards
- Drill-down capabilities to explore granular performance
- Supports strategic and operational decisioning
- Built in industry standard analytics tool (Power BI)

## Getting Started

To explore this dashboard locally:

1. **Install Power BI Desktop** (if not already installed)
2. Open 'PepsiCo_1.pbix'
3. Ensure the related datasets are accessible and linked in the Power BI model
4. Interact with the visuals, filters, and pages to explore insights

## Data

Place your data files inside the 'PepsiCo Datasets' folder. Typical tables expected by the dashboard:

- Sales transactions by SKU, region, period
- Product master with categories
- Profitability measures by channel
- Region hierarchy and mapping

> Ensure column names and types align with the existing Power BI model or update the model accordingly.

## Dashboard
